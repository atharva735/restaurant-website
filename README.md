# Saffron & Ember — Restaurant Website

A fine dining restaurant website built with HTML, CSS, and JavaScript.

## Features
- Menu with category filters and pricing
- Chef's Specials section
- Table reservation form with validation
- Smooth scroll navigation
- Fully responsive design

## How to Run
Open `index-5.html` in any browser — no server needed.
